Static Combo Handler

A static combo handler in NodeJS. If you want to use TaoBao-Style combo-hander, please visit here.

Install

npm i static-combo-handler

Usage

Express = require 'express'
combo = require 'static-combo-handler'
app = Express()
opt = {
    "port": 3333,
    "assetsPath": "./path-to-static-files-dir",
    "combo_tag":"??",
    "combo_dir_tag":"__",
    "combo_mod_split":","
}
app.set 'port', process.env.PORT or opt.port
app.use combo(opt)
app.listen app.get('port')

Deploy server to production environment

Step 1: Install PM2

npm install pm2 -g

PM2 is a production process manager for Node.js applications with a built-in load balancer. It allows you to keep applications alive forever, to reload them without downtime and to facilitate common system admin tasks.

Step 2: Install node module

npm install

Step 3: Start combo server in the production environment

sh bin/run.sh

Restart the server

sh bin/restart.sh

Stop the server

sh bin/stop.sh
